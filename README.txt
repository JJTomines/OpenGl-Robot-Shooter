HOW TO COMPILE
-Open the OpenGLProject.sln file using Visual Studio.
-Press debugger button to run project.

CONTROLS
a: Start Animation 
r: Reset Animation
left arrow key: move cannon left
right arrow key: move cannon right
f: Fire cannon
v: Change Camera
q,Q,esc: Quit Game
